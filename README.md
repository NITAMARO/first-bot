# first-bot

Un bot Telegram simple qui répond à la commande /start avec un message de bienvenue.

## Installation

```bash
pip install python-telegram-bot
```

## Utilisation

Remplace "TOKEN" dans bot.py par ton token Telegram, puis lance:

```bash
python bot.py
```
